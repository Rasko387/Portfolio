Objectif du projet

Multiburo est un projet web développé autour d’une entreprise fictive spécialisée dans la location d’espaces de coworking. L’application permet aux utilisateurs de consulter les espaces disponibles et de réserver un poste de travail en ligne. Elle intègre également un système de gestion des comptes avec différents rôles, notamment des utilisateurs et un administrateur. L’objectif du projet est de proposer une solution simple de réservation et de gestion d’espaces de coworking.

Fonctionnalités principales

Consultation des espaces

Visualisation des espaces de coworking disponibles avec leurs informations.

Réservation en ligne

Possibilité pour les utilisateurs de réserver un espace directement via l’application.

Gestion des comptes

Système d’authentification avec comptes utilisateurs et administrateur.

Base de données Firebase

Utilisation d’une base de données NoSQL pour stocker les données des utilisateurs et des réservations.

Bilan du projet

Ce projet m’a permis de découvrir le framework React ainsi que le fonctionnement d’une base de données NoSQL avec Firebase. Le travail en groupe a représenté un défi, notamment en raison de la prise en main de React que je ne maîtrisais pas encore complètement. Cette expérience m’a permis de mieux comprendre l’organisation d’un projet web et l’importance de structurer correctement son application afin d’éviter des problèmes techniques. Ce projet a été formateur sur le plan technique et organisationnel.
